# ConfigurationDemo

## 核心组件包

- Microsoft.Extensions.Configuration.Abstractions
- Microsoft.Extensions.Configuration

## 配置框架

- 以Key-Value字符串键值对的方式抽象了配置
- 支持从各种不同的数据源读取配置

## 核心类型

- IConfiguration
- IConfigurationRoot
- IConfigurationSection
- IconfigurationBuilder

## 扩展点

- IConfigurationSource
- IConfigurationProvider
